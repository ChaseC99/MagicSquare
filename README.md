# MagicSquare
Generate a [magic square](https://en.wikipedia.org/wiki/Magic_square).

Given an odd number `n`, this code will generate an n x n square 
where the sum of all of columns, rows, and the diagonals are equivalent.  

Example:
```
n = 5
|----|----|----|----|----|
|17  |24  |1   |8   |15  | => 65
|----|----|----|----|----|
|23  |5   |7   |14  |16  | => 65
|----|----|----|----|----|
|4   |6   |13  |20  |22  | => 65
|----|----|----|----|----|
|10  |12  |19  |21  |3   | => 65
|----|----|----|----|----|
|11  |18  |25  |2   |9   | => 65
|----|----|----|----|----|

 65   65   65   65   65  
```
